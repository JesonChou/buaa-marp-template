## Slide 4:

### 1.1 项目定位——什么是 BUAA Marp 模板项目

> 布局：B  |  内容：散文+有序列表(含<strong>子标题)  |  密度：ol li 22px lh3, --text-p-size:26px lh2

BUAA Marp 模板项目是一套基于 Marp 的北航主题演示文稿生成系统。用户只需提供文字、图片和表格数据，AI 即可自动完成排版，生成专业美观的 PPT。

**项目特色：**

1. 纯 Markdown 编写，专注内容创作，告别拖拽排版
2. 5 种页面类型：封面、目录、正文、过渡、结尾
3. 11 种正文布局，自动匹配内容形态
4. 集成 Awesome-Marp 增强组件库
5. 内置 AI Skill，一句话生成完整 PPT

**核心原则：**

用户提供完整素材，AI 负责辅助排版；大纲中没有的内容，AI 一律不自行添加。

## Slide 5:

### 1.2 工作流程——从大纲到 PPT

> 布局：G  |  内容：纵向流程图(5步)  |  密度：--fc-font-size:28px

流程图（纵向）：
  → 用户按模板编写大纲（填入文字、图片路径、表格数据）
  → AI 逐页分析 {页面类型, 内容类型, 布局选择, 密度参数}
  → AI 生成 presentation.md（套用 buaa.css 主题样式）
  → 用户在 VS Code 中预览（Ctrl+K V），微调 adj-* 占位符
  → 导出为 PDF / PPTX / HTML

## Slide 9:

### 2.2.1 Markdown 标题层级规范——一级与二级标题

> 布局：B  |  内容：散文段落  |  密度：--text-p-size:26px lh1.7

一级标题（`#`）：仅用于标识整体题目，一份大纲中最多出现一个。二级标题（`##`）：统一格式为 `## Slide N: {可选短标签}`，用于区分不同页码，标签仅在过渡页使用。

## Slide 10:

### 2.2.2 Markdown 标题层级规范——三级标题与深度限制

> 布局：C  |  内容：散文+居中表格  |  密度：--text-p-size:26px lh1.7

`###` 为正文主标题，格式 `### {编号} 主标题——副标题`。

| 规则 | 说明 |
| :--: | :--: |
| 有编号 | 标题中标注了序号 → badge 中填入该序号 |
| 无编号 | 未标注 → AI 根据原则自动分配 |
| 标题栏 | 标题栏中**不得出现序号**，序号统一在 badge 内 |

标题深度严格限制为三级（1 / 1.1 / 1.1.1），禁止四级及以上。`####` 仅用于同编号拆分多页的子页标识。

## Slide 34:

### 5.4 常用微调——adj-* 占位符

> 布局：B  |  内容：散文+callout  |  密度：--text-p-size:24px lh1.7

adj-* 是 CSS 注释占位符，不在 HTML 中激活。取消注释可微调：adj-img（图片高度）、adj-txt（文本字号行距）、adj-tbl（表格字号）、adj-list（列表字号行距）、adj-buaa-callout（callout 字号行距）。⚠️ adj-* 为 class 选择器在 scoped 中大概率不生效，应改用 `section { --var }` 注入 CSS 变量。
